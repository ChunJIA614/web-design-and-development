# Web Design and Development Project

## Overview
This repository contains the collaborative project for web design and development. The project is divided into multiple sections, each managed by a different team member. The goal is to create a comprehensive and user-friendly website with various features and pages.

## Project Structure
The project is organized into the following directories:

### WDD - CHAW CHUN JIA
- **HTML Files**: Contains product-related pages such as bedroom, kitchen, and living room products.
- **CSS Files**: Stylesheets for product pages.
- **JavaScript Files**: Scripts for product details.
- **Assets**: Icons and images for product pages.

### WDD - GOH CHUN WEN
- **HTML Files**: Pages for contact us, location, and terms.
- **CSS Files**: Stylesheets for the respective pages.
- **Branches**: Contains branch-specific pages and styles.
- **Assets**: Icons and images for branch pages.

### WDD - HIEW LI WEI
- **HTML Files**: Pages for order history, payment, and shopping cart.
- **CSS Files**: Stylesheets for the respective pages.
- **JavaScript Files**: Scripts for payment and shopping cart functionality.
- **Assets**: Icons and images for the respective pages.

### WDD - OOI JUN KANG
- **HTML Files**: Pages for about us, career, FAQ, and home.
- **CSS Files**: Stylesheets for the respective pages.
- **Assets**: Icons and images for the respective pages.

### WDD - YE LWIN AUNG
- **HTML Files**: Pages for login, signup, and profile.
- **CSS Files**: Stylesheets for the respective pages.
- **JavaScript Files**: Scripts for login, signup, and profile functionality.
- **Assets**: Icons and images for the respective pages.

## How to Use
1. Clone the repository to your local machine.
2. Open the project in your preferred code editor.
3. Navigate to the respective directories to view or edit the files.
4. Open the HTML files in a browser to preview the pages.
